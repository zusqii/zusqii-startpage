# 🌑 zusqii-startpage

A minimalist, high-performance glassmorphic startpage for Zen Browser.

## 🚀 Installation

### For Linux Users (Git)
1. **Clone the repository:**
   ```bash
   git clone https://github.com/zusqii/zusqii-startpage.git ~/.config/zusqii-startpage
   ```
2. **Set as Homepage:**
   `file:///home/zusqii/.config/zusqii-startpage/index.html`

### Quick Setup (General)
1. **Download** the project folder to your local machine.
2. **Edit links:** Open `index.html` to add or change your own links.
3. **Browser Extension:** Use an extension like *Custom New Tab URL* (Chrome) or *New Tab Override* (Firefox).
4. **Set Path:** Point the extension to your local `index.html` file.

---

### 🛠 Customization
- **Fonts:** Cleaned Noto Sans is included in the `fonts/` folder.
- **Style:** Edit `css/glass.css` to change the blur or transparency.
- **Dependencies:** None. No build tools required.
